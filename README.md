
### Run your tests

运用nvm进行node的版本管理。

nvm install latest            // 安装最新版的nodejs，会同时一起安装对应的npm

nvm install 8.11.3            // 安装指定版本号的nodejs，会同时一起安装对应的npm

nvm ls                        // 查看当前已安装的所有nodejs版本

nvm use 8.11.3                // nodejs版本切换
